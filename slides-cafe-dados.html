Parte 1

IA para Iniciantes

Sem jargão. Sem medo. Do zero ao primeiro agente.

O que é Inteligência Artificial?

🧠

Não é magia

IA são sistemas que aprendem padrões a partir de dados — como uma criança que aprende a reconhecer gatos vendo muitas fotos.

📊

Aprendizado de Máquina

Algoritmos que melhoram automaticamente com a experiência. Quanto mais dados, melhor o resultado.

💬

LLMs (Modelos de Linguagem)

Modelos treinados em textos da internet. Eles "completam" texto de forma coerente — o ChatGPT é um exemplo.

Como um LLM funciona?

# Você manda uma pergunta... user: "Explique o que é LangGraph em 1 frase" # O modelo prevê a próxima palavra, # uma de cada vez, baseado em probabilidades: assistant: "LangGraph" → "é" → "uma" → "biblioteca" → "para" → "criar" → "aplicações" → "com" → "agentes" → "de" → "IA" → "usando" → "grafos."

Não existe "raciocínio" — o modelo apenas prevê o próximo token (palavra/pedaço de palavra) mais provável.

Como palavras viram números?

Cada palavra é convertida em um vetor — uma lista de números que representa seu significado no espaço semântico.

Palavras com significados similares ficam próximas no espaço vetorial.
"rei - homem + mulher ≈ rainha" — operações matemáticas sobre vetores!

De LLM para Agente

LLM Puro

  • Só responde com o que foi treinado
  • Não pode buscar informações novas
  • Não pode executar ações
  • Não tem memória entre chamadas
  • Alucina quando não sabe

Agente de IA

  • Pode usar ferramentas (buscar, calcular)
  • Toma decisões (qual ferramenta usar?)
  • Tem memória de conversas anteriores
  • Executa múltiplos passos sozinho
  • Planeja antes de responder

Agente = LLM + Ferramentas + Memória + Tomada de Decisão

Anatomia de um Agente

1

Percebe

Recebe input do usuário e contexto

2

Decide

Escolhe o que fazer: responder, buscar, calcular

3

Age

Executa a ação com ferramentas

4

Observa

Recebe o resultado da ação

5

Responde

Retorna a resposta final ao usuário

Esse ciclo se repete até o agente decidir que tem uma resposta satisfatória.

Exemplo: Agente com Ferramentas

# Usuário pergunta: user: "Qual a cotação do dólar hoje?" # O LLM percebe que não sabe a cotação atual. # Mas ele tem acesso a uma ferramenta: buscar_na_web("cotação dólar") # Então ele decide usar a ferramenta: agent: tool_call(buscar_na_web, "cotação dólar hoje 27/06/2026") # A ferramenta retorna: "R$ 5,47" # Agora o LLM tem a informação e responde: assistant: "A cotação do dólar hoje (27/06) está em R$ 5,47."

Por que Múltiplos Agentes?

🎯

Especialização

Cada agente é expert em uma área. Um copywriter escreve melhor que um generalista.

⚡

Paralelismo

Vários agentes trabalham ao mesmo tempo. Resultado mais rápido.

✅

Qualidade

Um agente revisor checa o trabalho do copywriter. Menos erros.

🔄

Flexibilidade

Adicione ou remova agentes conforme a tarefa. O sistema se adapta.

RAG: Como dar superpoderes ao LLM

RAG (Retrieval Augmented Generation) combina busca em documentos com LLMs para respostas precisas e fundamentadas.

Sem RAG: LLM alucina. Com RAG: LLM responde com base em dados reais.

Parte 2

Introdução ao LangGraph

A biblioteca que transforma agentes em grafos

O que é LangGraph?

LangGraph é uma biblioteca Python para criar grafos de agentes com estado. Em vez de escrever uma função gigante com if/else, você define nós (etapas) e arestas (transições entre etapas) como um fluxograma que executa de verdade.

🗺️ Grafo Explícito

O fluxo do agente é visível e auditável. Dá até pra gerar diagrama automático.

💾 Checkpoints

Estado salvo automaticamente. Se falhar, retoma de onde parou.

📡 Streaming

Cada passo emite eventos. O frontend recebe atualizações em tempo real.

Conceitos Fundamentais

State (Estado)

Um dicionário que carrega todas as informações entre os nós.

class AgentState(TypedDict): messages: list route: str tools: list response: str

Nodes (Nós)

Funções Python que recebem o estado e retornam atualizações.

def classify_route(state): msg = state["messages"][-1] if "complexo" in msg: return {"route": "complex"} return {"route": "simple"}

Edges (Arestas): conectam nós. Podem ser fixas ou condicionais (roteamento).

Conditional Edges & Checkpoints

Conditional Edges (Roteamento)

# Depois de classificar, decide pra onde ir: graph.add_conditional_edges( "classify_route", # nó de origem decide_route, # função de decisão { "simple": "respond_fast", # se rota=simple → respond_fast "medium": "respond_medium", "complex": "squad_fanout" # se rota=complex → time de agentes } )

Checkpoints (Persistência)

LangGraph salva o estado após cada nó. Se o processo falhar no meio, você retoma exatamente de onde parou. Funciona com SQLite, Postgres ou qualquer storage.

Paralelismo com Send API

Código equivalente — declarativo, sem gerenciar threads:

from langgraph.types import Send def fanout_agents(state): # Dispara 3 agentes em paralelo return [ Send("run_agent", {"agent": "copywriter", "task": state["task"]}), Send("run_agent", {"agent": "strategist", "task": state["task"]}), Send("run_agent", {"agent": "reviewer", "task": state["task"]}), ] def run_agent(state): result = call_llm(state["agent"], state["task"]) return {"results": [result]} def consolidate(state): final = "\n\n".join(state["results"]) return {"response": final}

Passo 2: Definindo o Estado

from typing import TypedDict, Annotated, Literal from langgraph.graph.message import add_messages from langchain_core.messages import BaseMessage class AgentState(TypedDict): # Mensagens da conversa (com reducer add_messages) messages: Annotated[list, add_messages] # Rota decidida pelo classificador route: str # Sentimento do usuário sentiment: str # Nível de complexidade complexity: str # Resultado dos agentes do squad squad_results: Annotated[list, reducer_merge] # Resposta final para o usuário response: str

Tip: add_messages faz append automático de mensagens. Annotated define reducers para cada campo.

Passo 3: Nó de Classificação

import re ROUTES = { "squad": ["mega brain", "overdrive", "modo deus", "squad", "todos os agentes"], "premium": ["deep research", "análise profunda"], "standard": ["analise", "estratégia", "brain"], } def classify_route(state: AgentState) -> AgentState: msg = state["messages"][-1].content.lower() for route, keywords in ROUTES.items(): if any(kw in msg for kw in keywords): return {"route": route} # Análise de complexidade por heurística words = len(msg.split()) if words > 200: return {"route": "premium"} return {"route": "fast"} # default

Passo 4: Nós de Resposta

MODELS = { "fast": "google/gemini-2.5-flash-lite", "standard": "openai/gpt-4o", "premium": "anthropic/claude-sonnet-4", } async def respond(state: AgentState) -> AgentState: route = state["route"] model_id = MODELS.get(route, MODELS["fast"]) messages = state["messages"] # Prepara system prompt com contexto system = get_system_prompt(route) full_messages = [system] + messages # Chama o LLM via OpenRouter response = await call_openrouter( model=model_id, messages=full_messages, tools=TOOLS, # search, calculate, memory ) return {"response": response, "messages": [response]}

Passo 5: Squad Fan-Out

AGENTS = { "copywriter": "Escreva copy persuasiva para: ", "strategist": "Crie estratégia de marketing para: ", "social": "Adapte para redes sociais: ", "reviewer": "Revise e melhore este texto: ", } def fanout_agents(state: AgentState): task = state["messages"][-1].content return [ Send("run_agent", {"agent": name, "task": task}) for name in AGENTS ] async def run_agent(state: AgentState) -> AgentState: agent = state["agent"] task = state["task"] prompt = AGENTS[agent] + task result = await call_openrouter( model="openai/gpt-4o-mini", messages=[{"role": "user", "content": prompt}] ) return {"squad_results": [{"agent": agent, "result": result}]}

Passo 6: Consolidar Resultados

async def consolidate(state: AgentState) -> AgentState: results = state["squad_results"] # Junta resultados de todos os agentes combined = "\n---\n".join( f"### {r['agent'].upper()}\n{r['result']}" for r in results ) # Passa pelo copy-chief para unificar final = await call_openrouter( model="anthropic/claude-sonnet-4", messages=[{ "role": "system", "content": "Você é o Copy Chief. Consolide os trabalhos dos agentes em uma resposta final coesa." }, { "role": "user", "content": combined }] ) return {"response": final, "messages": [final]}

Passo 7: Montando o Grafo

from langgraph.graph import StateGraph, START, END from langgraph.checkpoint.memory import MemorySaver def decide_route(state: AgentState) -> str: return state["route"] # "fast" | "standard" | "premium" | "squad" def build_graph() -> StateGraph: graph = StateGraph(AgentState) # Adiciona nós graph.add_node("classify", classify_route) graph.add_node("respond", respond) graph.add_node("fanout", fanout_agents) graph.add_node("run_agent", run_agent) graph.add_node("consolidate", consolidate) # Conecta graph.add_edge(START, "classify") # Roteamento condicional graph.add_conditional_edges( "classify", decide_route, { "fast": "respond", "standard": "respond", "premium": "respond", "squad": "fanout", } ) graph.add_conditional_edges( "fanout", lambda s: [ Send("run_agent", {"agent": a, "task": s["messages"][-1].content}) for a in AGENTS ] ) graph.add_edge("run_agent", "consolidate") graph.add_edge("consolidate", END) graph.add_edge("respond", END) return graph.compile(checkpointer=MemorySaver())

Executando o Mega Brain

import asyncio from graph import build_graph async def main(): app = build_graph() # Configuração da execução config = {"configurable": {"thread_id": "session-123"}} # Streaming: cada evento mostra o progresso async for event in app.astream( {"messages": ["Crie uma campanha de marketing para o Café com Dados"]}, config, stream_mode="updates" ): node_name = list(event.keys())[0] print(f"✓ {node_name} concluído") # Resultado final state = await app.aget_state(config) print("\n=== RESPOSTA FINAL ===\n") print(state.values["response"]) asyncio.run(main())
