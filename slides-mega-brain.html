    Router Inteligente

    O "segredo" do Mega Brain é só um if/else bem feito

    # router.py — O "Mega Brain" é só roteamento
    def route(text: str, sentiment: str) -> str:
        if "mega brain" in text.lower() or sentiment == "alta":
            return "claude-sonnet-4-20250514" # Topo de linha
        elif "brain" in text.lower() or sentiment == "media":
            return "gpt-4o" # Intermediário
        else:
            return "gemini-2.0-flash-lite" # Econômico

    # Copy Chief Squad só é ativado em "mega brain"
    if model == "claude-sonnet-4-20250514":
        squad.activate("copy-chief") # 7 agentes

    A mágica é arquitetura, não feitiço. O "Mega Brain" é routing + squad maior.

    # squads/copy-chief/squad.yaml
    name: copy-chief
    description: Squad de copywriting multi-agente
    agents:
      - name: copy-chief
        role: Coordenador — recebe, delega, revisa
        model: claude-sonnet-4
      - name: strategist
        role: Estratégia — posicionamento, público, CTA
        model: gpt-4o
      - name: copywriter
        role: Redação — headlines, body, storytelling
        model: claude-sonnet-4
      - name: social-media
        role: Adaptação por plataforma
        model: gpt-4o-mini
      - name: designer
        role: Briefing visual
        model: gemini-2.0-flash

    Cada agente é independente, com modelo, ferramentas e instruções específicas

    O "JARVIS" é só um pipeline com stages

    01
    Voice Input
    Whisper STT transcreve áudio → texto

    Classifica intenção: "normal", "brain" ou "mega brain"
    03
    Model Router
    Roteia para Gemini Flash, GPT-4o ou Claude Opus
    04
    Copy Chief Squad
    7 agentes em paralelo: strategist, copywriter, social, designer, seo, reviewer
    05
    Dashboard
